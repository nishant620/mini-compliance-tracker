Mini Compliance Tracker to manage tasks for multiple clients.

2. Tech Stack
    Frontend: React (Vite)
    Backend: Node.js, Express
    Database: MongoDB

3. Features
- View clients
- View tasks per client
- Add tasks
- Update task status
- Filter tasks
- Highlight overdue tasks

4. Setup Instructions
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev

5. Assumptions
- Clients are pre-created
- No authentication required

6. Tradeoffs
- Kept UI simple for faster development
- No advanced styling
